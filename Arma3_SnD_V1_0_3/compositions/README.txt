To import these compositions

1: Find the directory of your compositions

	#:\Users\#\Documents\Arma 3 - Other Profiles\#\compositions

	Ei C:\Users\YOURNAME\Documents\Arma 3 - Other Profiles\Kerry\compositions

2: Copy the folders from  Arma3_SnD_V#_#_#\compositions folder into your compositions folder


To use the compositions

0: Ensure that in Settings (top left) > Preferences > Automatic Composition Layering > Off, to prevent the composition from breaking

1: Press F2 and select Custom on the right hand side of the editor

2. Place the composition like an entity
